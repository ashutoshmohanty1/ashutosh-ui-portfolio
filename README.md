# 🌐 Ashutosh UI Portfolio

A modern and responsive personal portfolio website showcasing my projects, skills, and UI design capabilities.

## 🚀 Features

- ✨ Clean and modern UI design
- 🎨 Black & purple theme
- 🪐 Animated orbit profile image
- 🎠 Project carousel with smooth transitions
- 📱 Fully responsive (mobile + desktop)
- ⚡ Interactive animations (hover, tilt, parallax)
- 📩 Contact via email integration

## 🛠️ Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Firebase Hosting
